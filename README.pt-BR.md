## 💡 DICAS RÁPIDAS

### Copiar Token
1. Abra app
2. Modal aparece
3. Clique no botão copiar
4. Token salvo no clipboard

### Resetar Tudo
1. F12 (DevTools)
2. Application → LocalStorage
3. Clique em "profisstoken" → Delete
4. Refresh página

### Mudar Cores
1. Abra `src/App.css`
2. Procure por `:root`
3. Mude as cores (ex: `--primary: #2563eb;`)

### Adicionar Logos
