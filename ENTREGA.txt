🔌 ENDPOINTS DISPONÍVEIS

  TOKENS (4)
  ├─ POST /api/generate-token      → Gerar novo token
  ├─ POST /api/validate-token      → Validar acesso  
  ├─ POST /api/restore-token       → Recuperar em novo device
  └─ POST /api/delete-token        → Deletar token

  PAGAMENTOS (5)
  ├─ POST /api/create-payment      → Criar pagamento PIX
  ├─ GET  /api/payment/:id         → Status do pagamento
  ├─ GET  /api/payments            → Listar pagamentos
  ├─ POST /api/confirm-payment     → Confirmar pagamento
  └─ POST /api/refund-payment      → Reembolsar

  ADMIN (3)
  ├─ GET  /api/statistics          → Estatísticas
  ├─ GET  /api/health              → Status da API
  └─ POST /api/webhooks/mercado-pago → Handler MP

🚀 COMO COMEÇAR

  1. Leia DEPLOY.md para instruções passo-a-passo

  2. Prepare a hospedagem:
     • Crie banco de dados profissapp
     • Execute script database.sql

  3. Configure credenciais:
     • Copie .env.example → .env
     • Preencha Mercado Pago keys

  4. Upload dos arquivos:
     • Via FTP/SFTP
     • Ou via SCP/SSH

  5. Teste a API:
     • curl https://seu_dominio.com/api/health
     • curl -X POST https://seu_dominio.com/api/generate-token

  6. Configure webhook:
     • Mercado Pago Dashboard
     • URL: https://seu_dominio.com/api/webhooks/mercado-pago.php

  7. Integre com frontend:
     • Usar hooks useAuth e usePayment
     • Componente PaymentModal pronto
